# YIBO.AI - Personal Portfolio

个人作品集网站，展示AI产品经理项目经验和技术能力。

## 项目特色

- 🎨 现代设计风格，深色主题
- 📱 响应式设计，支持移动端
- 🖼️ 图片滑动交互，支持放大查看
- 🚀 基于Vite构建，快速开发
- 🌐 GitHub Pages部署支持

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: CSS3 + 自定义动画
- **部署**: GitHub Pages

## 项目结构

```
yibo-ai/
├── index.html          # 主页面
├── package.json        # 项目配置
├── src/                # 源代码目录
│   ├── components/     # React组件
│   ├── App.tsx         # 主应用组件
│   └── main.tsx        # 入口文件
├── images/             # 项目图片资源
└── README.md           # 项目说明
```

## 本地开发

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 部署到GitHub Pages

### 方法1：手动部署

1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择 `gh-pages` 分支或 `docs` 文件夹作为源

### 方法2：使用GitHub Actions自动部署

项目已配置 `.github/workflows/deploy.yml`，提交代码到main分支后会自动部署。
